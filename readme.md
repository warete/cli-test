# CLI app test

Небольшая библиотека для реализации консольных приложений + тестовое приложение для демонстрации.

Поддерживает аргументы с фигурными скобками и без, т.к. оболочки bash и sh сами парсят множественные значения. С параметрами аналогично. 

**В zsh приложение не работает из-за другой логики обработки фигурных скобок.**

В проекте добавлены следующие dev-зависимости:
- cs-fixer для форматирования: `composer fixcs`
- phpstan на уровне 9: `composer phpstan`
- phpunit: `composer test`. Основная часть библиотеки покрыта тестами

Старт приложения:
```bash
composer i
php app.php test {verbose,overwrite} [log_file=app.log] {unlimited} [methods={create,update,delete}] [paginate=50] {log}
```
